## Topics Covered

- **Arrays & Strings**: Two pointers, sliding window, sorting
- **Linked Lists**: Fast & slow pointers, reversal, merge
- **Trees & Graphs**: DFS, BFS, traversals
- **Dynamic Programming**: Memoization, tabulation
- **Hash Tables**: HashMap, HashSet
- **Stack & Queue**: Monotonic stack, priority queue
- **Binary Search**: Search space reduction
- **Recursion & Backtracking**: Combination, permutation

## How to Use

1. Clone the repository:
```bash
git clone https://github.com/hectorbiminchumo/coding-challenges.git
cd coding-challenges
```

2. Navigate to any problem:
```bash
cd arrays/easy/001-two-sum
```

3. Run the solution:
```bash
# JavaScript
node solution.js
